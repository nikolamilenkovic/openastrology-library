## FAQ

**Q: Can I use this in my open source project?**
A: Yes! Use it freely under AGPL-3.0. Your project must also be AGPL/GPL licensed.

**Q: Can I use this in my commercial SaaS product?**
A: Yes, but you need a professional Swiss Ephemeris license from Astrodienst AG. Once you have that, you can use this library under LGPL-3.0.

**Q: Do I need to pay the openastrology-library author?**
A: No! This library itself is free. You only need to purchase a Swiss Ephemeris professional license from Astrodienst AG if you want to use it commercially.

**Q: What if I just want to use it internally in my company?**
A: If employees or clients access it via a network (web app, API), AGPL-3.0 requires you to provide the source code to users. For proprietary use, get a Swiss Ephemeris professional license and use this library under LGPL-3.0.

**Q: Can I modify the library?**
A: Yes, under both licenses. AGPL-3.0 requires you to share all modifications. LGPL-3.0 requires you to share modifications to this library only (not your application code).
